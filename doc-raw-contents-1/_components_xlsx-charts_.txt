this component allows importing excel charts into your documents. in simple words, create your charts in excel because excel is built for this (among others) and use this component to get it into your documents as it will be shown below. combined with component, this one provides the full support to share structured data in a professional way, including some extent of interactivity. currently we support only: line charts column and bar charts pie and doughnut charts we do not support importing the colours you assign to the series in the excel chart. the colours that will be used are randomly selected in order to provide the best user experience possible. the next examples are based on the files located as shown below: observe that the excel files are located in the same folder as the parent document. these files can be also located in a sub-folder of the parent document folder, but not outside the parent document folder. : the xlsx file. other excel formats (older) are not accepted. note that when provided as simple file name, the component search for it in the same folder where the its parent document is located. otherwise it should be provided as relative path from the folder where the its parent document is located. : the sheet where the chart is located inside the workbook : the chart title : the parent document, including its relative path to the root of the site directory. : specify if the chart will be wrapped into a container having a thin border. the default value is so, if you donâ€™t want a border, you can pass this parameter with value or to ignore it and do not pass anything
