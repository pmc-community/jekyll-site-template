custom annotations are comments added by the user to parts of the documents. can be seen and used only by the user who created them. working with custom annotations is possible only from the document body. note the difference between a custom note and a custom annotation is that a note is connected to a document, while annotations are connected to parts or placeholders inside a document. limitations for the moment, annotations can be added only in the desktop variant of the site. sharing taxonomies makes that annotations to be possible to be seen and updated on mobile variant too.
